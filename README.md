# 澳門統計數據

## 澳門政府公職人員（公務員）薪俸點換算

|年份|每點薪俸點為澳門元|
|---|---|
|1999|50|
|1999|55|
|2008|59|
|2012.5|66|
|2013.5|70|
|2014.5|74|
|2015|78|
|2016|81|
|2017|83|
|2018|85|
|2019|88|
|2020|91|

由回歸前每個薪俸點50元增加至目前的59元（2008年1月1日起由55元增加至59元）
